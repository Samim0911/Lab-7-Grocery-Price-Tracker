Grocery Price Tracker:

~ Description:
This program reads grocery items and prices from a text file, stores them in arrays, calculates the average price and writes the results to a new file.

~ How to Run:
1. Put `GroceryTracker.java` and `groceries.txt` in the same folder
2. Compile:
   javac GroceryTracker.java
3. Run:
   java GroceryTracker
4. Check `grocery_report.txt` for the output

~ Example Input (groceries.txt)
Apples,1.49
Bread,2.79
Milk,3.19
Eggs,2.29
Cheese,4.99

~ Example Output (grocery_report.txt)
Grocery Items Report
----------------------
Apples: $1.49
Bread: $2.79
Milk: $3.19
Eggs: $2.29
Cheese: $4.99

Average Price: $2.95
